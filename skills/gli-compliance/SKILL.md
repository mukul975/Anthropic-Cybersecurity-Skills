# GLI Compliance Skill

Validate Gaming Laboratories International standards compliance.

## GLI-19: Interactive Gaming Systems v3.0

Online/mobile casino platforms: system security, player accounts, game integrity, RNG, transactions, responsible gaming.

## GLI-33: Event Wagering Systems

Sports betting platforms: wagering integrity, geolocation, age verification, odds management, fraud detection.

## GLI-GSF: Gaming Security Framework

CIS Controls-based gaming security with Implementation Groups (IG1-IG3).

## Platform Systems

- **[Sportsbook Platform]**: GLI-33 - Spring Java, React Native, Cloudflare
- **[Casino Platform]**: GLI-19

## Output Format

1. System and applicable standard
2. Compliance matrix per section
3. Technical security assessment
4. Certification status
5. Remediation plan
